**Спринт 3 — DSL до полноценного (≈ 1 неделя):**

## Шаблон нового плана

Каждый план должен содержать разделы:

1. **# Этап N — Название**
2. **## Контекст** — почему делаем, какую проблему решаем
3. **## Синтаксис / YAML** — как фича выглядит для пользователя
4. **## Хранилище / SQL** — изменения в схеме БД (если применимо)
5. **## Изменения в коде** — конкретные файлы и сигнатуры
6. **## Тесты** — какие тесты добавляются (юнит + integration)
7. **## Verification** — как проверить вручную end-to-end
8. **## Эстимейт** — оценка трудозатрат с разбивкой
